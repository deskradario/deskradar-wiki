Bill of Materials

Everything you need to build a DeskRadar.

This list is based on the current DeskRadar hardware build. Some enclosure and power distribution parts are custom or still need exact part names confirmed.

Core Electronics

QtyPartNotes
1Raspberry Pi 4BOther Pi models may work, but the current build uses a Pi 4B.
1Waveshare 64x64 2.5 pitch LED matrixCurrent firmware and config assume a 64x64 HUB75 matrix.
1Waveshare I2C LCD displayUsed by the Pi LCD service for status and closest aircraft text.
1FlightAware Pro StickADS-B receiver for the local dump1090 feed.
11090 MHz antennaADS-B antenna for the receiver.
1HUB75 ribbon cableConnects the MatrixPortal/display driver path to the LED matrix.
