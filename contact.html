Contact

Feature requests, bug reports, and code issues.

Email

For feature requests or bug reports, email deskradar@deskradar.io.

Reports sent by email will be reviewed before being added to the public feature and bug board.

GitHub Issues

If the feature or bug is code-related, feel free to raise an issue directly on GitHub:

  • deskradar issues — main Raspberry Pi application.
  • deskradar-pi-services issues — Pi services, LCD, configurator, and boot checks.
  • deskradar-matrixportals3 issues — MatrixPortal S3 firmware.
