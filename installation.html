Installation

Get DeskRadar running using the pre-built Raspberry Pi image.

Pi Image

The easiest way to get DeskRadar running is to flash the pre-built Pi image. It is intended to include the main app, Pi services, systemd units, NetworkManager AP profile, Avahi hostname setup, and the default config file at /etc/deskradar/config.json.

Download: Pi image link coming soon.

The image is based on Raspberry Pi OS Bookworm (64-bit). It has not yet been aggressively trimmed, so a smaller image may follow later.
