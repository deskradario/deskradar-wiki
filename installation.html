Installation

Get DeskRadar running using the pre-built Raspberry Pi image.

Pi Image

The easiest way to get DeskRadar running is to flash the pre-built Pi image. It includes all software pre-installed and configured — no manual dependency installation required.

Download DeskRadar Pi Image (link coming soon)

The image was built fresh from Raspberry Pi OS Bookworm (64-bit) but still contains a significant amount of default bloat. Removing unnecessary packages is recommended before deploying in production.

Flashing the Image

  1. Download the image once the image link is available.
  2. Flash it to a microSD card using Raspberry Pi Imager or dd.
  3. Insert the card into your Raspberry Pi 4 and power it on.

After First Boot

On first boot the Pi will broadcast a WiFi access point named deskradar (password: deskradaradsb). Connect to it from your phone or laptop and open http://10.42.0.50:5000 to configure your home WiFi network and set your location coordinates.

See Architecture for a full description of the boot sequence and network modes.

MatrixPortal S3 Firmware

The Adafruit MatrixPortal S3 firmware is installed separately before first use by copying the contents of deskradar-matrixportals3/src/ to the CIRCUITPY volume. The repo's install.sh currently assumes the macOS mount path /Volumes/CIRCUITPY/; on Linux, copy to the mounted CIRCUITPY path instead. See the Source page for the repository link and API details.
